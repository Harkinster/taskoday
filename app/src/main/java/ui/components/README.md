# Composants Compose fournis

Ce dossier contient une base à adapter au projet existant.

## Important

Les fichiers sont volontairement génériques.
Il faut probablement :
- changer les packages ;
- remplacer les imports d’icônes selon les dépendances présentes ;
- remplacer `mascotRes`, `wordmarkRes`, `avatarRes` par les vrais `R.drawable...` ;
- brancher les composants sur les modèles de données existants.
