## Direction artistique globale
- Univers fantasy / jeu vidéo / aventure
- Couleurs dominantes : bleu nuit, violet, néon bleu/cyan
- Effets lumineux et contours brillants
- Cartes avec bordures lumineuses
- Interface immersive mais lisible
- Style moderne, enfant/ado, un peu RPG
- Logo Taskoday visible dans le header
- Avatar visible en haut à droite
- Navigation en bas avec icônes

## Contraintes UX
- Garder une bonne lisibilité
- Éviter de surcharger
- Conserver l’esprit “jeu” sans perdre l’aspect pratique
- Les écrans doivent rester cohérents entre eux
